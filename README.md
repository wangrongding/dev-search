# dev-search

Quickly search for what you want from the command line!

在命令行中快速搜索你想要的！

## Install （安装）

```sh
npm i dev-search -g
```

## Usage（使用）

Quickly search for something.

迅速搜索一些东西。

```sh
ds <content>
```

![](https://assets.fedtop.com/picbed/202208311621730.gif)

快速打开文档

```sh
ds -d
```

![](https://assets.fedtop.com/picbed/temp2.gif)

快速打开地图

```sh
ds -m
```
